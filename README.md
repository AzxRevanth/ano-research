# Tasks upto now:
1. Data foundation — You pulled 2 months (Sept–Oct 2012) of AMPds2 data: whole-home mains (WHE: V, I, P, Q, S) plus 6 selected appliances (FRE, HPE, DWE, CWE, WOE, B1E) at 1-minute resolution, joined with hourly weather (temperature, humidity, wind, visibility, pressure, and a simplified weather regime like Clear/Cloudy/Rain/Fog).
2. Synthetic anomaly injection — Since AMPds2 has no real anomaly labels, you injected 92 synthetic anomalies across 5 distinct contextual scenarios (heating running on a warm day, night-time load spikes, a stuck appliance, power spikes with no weather justification, weekend behavior bleeding into a weekday) and logged the exact window + type for each. This gave you real, known ground truth instead of a guessed proxy.
3. Windowing — The full timeline was chopped into non-overlapping 15-minute windows (2,928 windows for 2 months), with each window containing all 11 sensor channels together — this is the critical fix we made, since the first attempt accidentally treated each sensor as its own separate timeline.
4. TSFresh extraction — For every window, TSFresh computed ~700+ statistical descriptors per sensor (mean, variance, FFT coefficients, entropy, trend, etc.) — this is your behavior vector, describing what the household was doing in each 15-min slice, independent of any label.
5. FRESH pruning — Using your real injected labels (not a proxy), select_features() ran hypothesis tests against every one of those thousands of features and kept only the ~20 that statistically distinguish anomalous windows from normal ones. This is your final, compact behavior vector.
6. Context vector — Built separately, no TSFresh involved: cyclic hour-of-day/day-of-week encodings, weather regime one-hot flags, and raw weather numerics — describing the situation the household was in during each window.
7. Final output — ampds_behavior_context_labeled_features.csv: one row per 15-min window, columns = pruned behavior features + context features + is_anomaly (0/1) + anomaly_type (which of the 5 scenarios, or 'normal').
